### 二. 图形选中和编辑
#### 1. 图形选中
点击图形上的点(必须是图形上，封闭图形内部的点无效), 被选中的图形会显示关键点(如圆心，线段端点，曲线控制点等)，可以删除，编辑，填充，平移等。
